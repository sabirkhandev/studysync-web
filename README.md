# 📚 StudySync – Online Education Platform Landing Page

A modern and fully responsive **Online Education Landing Page** built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. The project focuses on delivering a clean user interface, smooth animations, and a seamless user experience across mobile, tablet, and desktop devices.

## ✨ Key Features

- **Responsive Mobile Menu:** JavaScript-powered hamburger menu that opens on mobile devices and automatically closes when clicking outside the menu.
- **Smooth Entry Animations:** Custom CSS keyframe animations that smoothly slide content into view when the page loads.
- **Feature Grid Layout:** A responsive six-card feature section built with CSS Grid, using `nth-child` selectors to apply unique background colors to each card.
- **Interactive Testimonial Cards:** Beautiful testimonial cards with hover effects, including smooth scaling and shadow transitions for a more engaging user experience.
- **Newsletter Signup Section:** A responsive newsletter subscription section with a split layout that adapts perfectly across desktop and mobile devices.

## ⚙️ Installation & Setup

Clone the repository

```bash
git clone https://github.com/sabirkhandev/studysync-web.git
```

Go to the project directory

```bash
cd studysync-web
```

Run the project

```bash
Open index.html in your browser
```

Or use the **Live Server** extension in VS Code for a better development experience.
